A Google Maps directive for AngularJS

Some kind of blurb goes here.

Features

  • 12 directives for the most used Google Maps objects.
  • Load as many .geojson files as you want on the same map. Google Maps data layer only allows for one file.
  • Create custom controls and infowindows in raw HTML with whatever behavior you'd like.
  • Two way data binding for Google Map objects. Changes to markers and editable rectangles update your model accordingly.

Install

To install with Bower cd to your directory and enter the following:

bower install ng-maps

You can also directly download the package here.

Quickstart

All you need to get started is Google Maps, AngularJS, and ng-maps. Link to all of those three in your HTML in that order.

<script src='//maps.googleapis.com/maps/api/js'></script>
<script src="//ajax.googleapis.com/ajax/libs/angularjs/1.3.3/angular.min.js"></script>
<script src="bower_components/ng-maps/dist/ng-maps.js"></script>

Next, include ng-naps in your AngularJS app and you're set to jet.

var app = angular.module('App', ['ngMaps'])

To get started with a basic map, add this somewhere your HTML.

<map ng-transclude class="short-map"> </map>

Important: give your map a class with a height property in pixels or percentage. Because of how Google Maps is, each parent div needs a height if you choose to use a percentage height.
