ng-maps is package written by Will Leahy which makes it easy to write data intensive Google Maps applications.

ng-maps was written especially for making large amounts of GIS data interactive and customizable. It takes care of the dirty work and lets you focus purely on configuring your dataâ€™s behavior and appearance.

ng-maps is build on AngularJS, a JavaScript framework which enhances HTML. This allows you to build your map using custom HTML tags, which makes structuring and scaling your application easy as pie.

Setup Instructions

For a step-by-step guide on how to install and use ng-maps go to the install and setup. If you're more experienced with angular look to the quickstart below.

Technical Features

  • 12 directives for the most used Google Maps objects.
  • Load as many .geojson files as you want on the same map. Google Maps data layer only allows for one file.
  • Create custom controls and infowindows in raw HTML with whatever behavior you'd like.
  • Two way data binding for Google Map objects. Changes to markers and editable rectangles update your model accordingly.

Install

To install with Bower go to your directory in terminal and enter the following:

bower install ng-maps

You can also directly download the package

Quickstart

All you need to get started is Google Maps, AngularJS, and ng-maps. Link to all of those three in your HTML in that order.

<script src='//maps.googleapis.com/maps/api/js'></script>
<script src="//ajax.googleapis.com/ajax/libs/angularjs/1.3.3/angular.min.js"></script>
<script src="bower_components/ng-maps/dist/ng-maps.js"></script>

Next, include ng-maps in your AngularJS app and you're set to jet.

var app = angular.module('App', ['ngMaps'])

To get started with a basic map, add this somewhere your HTML.

<map ng-transclude class="short-map"> </map>

Give your map a class with a height property in pixels or percentage. Because of how Google Maps is, each parent div needs a height if you choose to use a percentage height.

Now visit the example page to see some directives in action or browse through the documentation for a more in depth explanation of this package.
