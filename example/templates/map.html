Creates a <div> containing a Google Map instance. Requires a height property. Other ng-data-map directives can be placed inside this directive. Requires a center parameter.

HTML 

<div map ng-transclude class="google-map" center="map.center" options="map.options"> </div>

JavaScript 

			$scope.map = {
			  center: [39, -121],
			  options: { 
				  zoom: 4,
				  streetViewControl: false,
				  scrollwheel: false
				}
			};
